# Draft Abstract

Financial fraud detection remains difficult because fraudulent behavior is rare, adaptive, and often embedded in transaction patterns rather than isolated records. This paper presents FRAPH, a graph-aware fraud analysis framework that combines a web-based analysis workflow with a comparative evaluation pipeline for both traditional machine learning models and a proposed graph neural network (GNN) baseline. The system ingests transaction CSV files, performs preprocessing and graph construction, generates dashboard-ready fraud analysis outputs, and evaluates `KNN`, `Logistic Regression`, `Linear SVC`, `Random Forest`, and a transaction-graph GNN under a unified repeated stratified evaluation protocol. On a PaySim-derived labeled transaction sample, `Random Forest` achieved the strongest overall performance with an F1-score of `0.9385`, ROC-AUC of `0.9823`, and PR-AUC of `0.9515`, while the proposed GNN achieved an F1-score of `0.2906`, ROC-AUC of `0.8476`, and PR-AUC of `0.3185`. These results show that the current GNN captures meaningful fraud-related structure and substantially improves recall relative to several linear baselines, but still trails the strongest tree-based model on this dataset. The framework contributes a reproducible experimental pipeline, cross-platform implementation, ablation support, and a practical foundation for extending fraud detection research toward graph-native datasets and stronger GNN architectures.
